# Portafolio Web — Sofía Reyes

Sitio web de portafolio personal para **Sofía Reyes**, diseñadora gráfica y visual. Desarrollado con HTML y CSS puro, sin dependencias ni frameworks externos.

## Descripción

Página web de una sola pantalla (one-page) que presenta el trabajo y perfil profesional de Sofía Reyes. Incluye navegación fija, sección hero, presentación personal, proyectos destacados y formulario de contacto.

## Estructura del proyecto

```
Portafolio-web/
├── index.html       # Archivo principal (HTML + CSS embebido)
└── README.md        # Documentación del proyecto
```

## Secciones

| Sección | Descripción |
|---|---|
| **Hero** | Presentación principal con nombre, rol y llamada a la acción |
| **Sobre mí** | Biografía breve, experiencia y lista de habilidades |
| **Proyectos** | Tres cards con trabajos destacados (branding, editorial, packaging) |
| **Contacto** | Formulario de contacto con campos de nombre, email, asunto y mensaje |

## Proyectos destacados

- **Nocturne Studio** — Identidad visual completa para estudio de fotografía
- **Raíz Magazine** — Dirección de arte y maquetación de revista cultural (80 páginas)
- **Botánica Cosméticos** — Diseño de empaque sostenible para línea orgánica

## Tecnologías

- HTML5 semántico
- CSS3 puro (variables CSS, Flexbox, media queries)
- Google Fonts: Playfair Display + Inter

## Diseño

- **Paleta de colores:** Negro `#0d0d0d`, Blanco `#ffffff`, Acento rojo `#E63946`
- **Tipografía:** Playfair Display (títulos) + Inter (cuerpo)
- **Responsive:** Adaptado para escritorio, tablet (768px) y móvil (480px)

## Mejoras pendientes

1. **Menú hamburguesa** para navegación en móvil (actualmente los links se ocultan en pantallas < 768px)
2. **Imágenes reales** — reemplazar los placeholders CSS por `<img>` con `srcset` y `loading="lazy"`
3. **Formulario funcional** — conectar el `<form>` a un servicio como [Formspree](https://formspree.io) o Netlify Forms
